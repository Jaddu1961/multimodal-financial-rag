## 🎯 What This Does

This system ingests Tesla financial PDF reports and answers questions about them using:
- **Text chunks** from the report
- **Table data** with financial figures
- **Chart descriptions** generated by a vision AI (LLaVA)

### Example Questions It Can Answer:
- *"What was Tesla's total revenue in Q3 2023?"* → **$23,350 million**
- *"What is Tesla's gross profit margin?"* → **17.9%**
- *"What do the charts show about revenue growth?"* → Detailed chart analysis

## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- Ollama (for vision)
- Groq API key (free)

### 1. Clone Repository
```bash
git clone https://github.com/Jaddu1961/multimodal-financial-rag.git
cd multimodal-financial-rag
```

### 2. Setup Python Environment
```bash
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Setup Environment Variables
```bash
cp .env.example .env
# Edit .env and add your keys:
# GROQ_API_KEY=your_key_here
# GEMINI_API_KEY=your_key_here (optional)
```

### 4. Install & Start Ollama
```bash
# Install from https://ollama.com
ollama pull llava
```

### 5. Start FastAPI Backend
```bash
uvicorn app.api.main:app --reload --host 0.0.0.0 --port 8000
```

### 6. Download & Ingest Tesla Report
```bash
curl -L -o data/raw/TSLA-Q3-2023.pdf \
"https://digitalassets.tesla.com/tesla-contents/image/upload/IR/TSLA-Q3-2023-Update-3.pdf"
```

Then go to `http://localhost:8000/docs` → `POST /ingest/upload`

### 7. Start React Frontend
```bash
cd frontend/react-app
npm install
npm run dev
```

Open `http://localhost:5173` 🎉

## 🔌 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/ingest/upload` | Upload & process PDF |
| `POST` | `/query/ask` | Ask a question |
| `GET` | `/documents/list` | List documents |
| `GET` | `/ingest/stats` | Vector store stats |
| `DELETE` | `/ingest/document/{id}` | Delete document |
| `GET` | `/health` | System health |
